# Event Management System

## Project Overview

Event Management System is a Java Swing based desktop application developed to manage events efficiently. The system provides a simple graphical user interface for adding, viewing, and searching events.

This project demonstrates the implementation of Object-Oriented Programming concepts, Java Swing GUI development, exception handling, and input validation.

---

## Features

* Login System
* Add Event
* View Events
* Search Event
* User-Friendly Dashboard
* Input Validation
* Exception Handling
* OOP Concepts Implementation
